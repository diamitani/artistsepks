# Site Types — Section Recipes
## Reference for webpage-designer SKILL.md

---

## SaaS Homepage (conversion-focused)

```
01. NAV          Sticky. Logo left, links center, CTA right. On scroll: add bg blur.
02. HERO         H1 (<10 words, outcome-focused) + sub (1-2 lines) + dual CTA + product mockup/screenshot
03. SOCIAL PROOF Logo bar ("Trusted by X companies") OR testimonial quote strip OR user count
04. PROBLEM      "You're stuck doing X when you should be doing Y." No product yet.
05. FEATURES     3-column icon grid OR alternating text+visual pairs (3 features max in v1)
06. HOW IT WORKS 3 numbered steps. Simple. Step → Action → Result.
07. SOCIAL PROOF Testimonial cards (3) OR video testimonial placeholder
08. PRICING      3 columns (Free / Core / Pro). Annual toggle. Feature checklist. CTA per tier.
09. FAQ          5-7 questions. Address top objections. Accordion optional.
10. FINAL CTA    Repeat offer. Urgency if warranted. "Start free" dominant.
11. FOOTER       4-col links. Newsletter. Social. Legal. Brand.
```

## Landing Page (single product/offer)

```
01. NAV          Minimal. Logo + single CTA. No distracting links.
02. HERO         Headline + sub + ONE CTA + hero image or demo
03. SOCIAL PROOF Compact: 3 logos or "X users" + star rating
04. BENEFIT LIST 3-5 bullet outcomes (not features). "You will..." language.
05. HOW IT WORKS 3 steps visual
06. TESTIMONIALS 2-3 quotes with photo + name + role
07. PRICING      Single offer or 2-tier. Clear value prop per.
08. CTA          Repeat + countdown/scarcity if relevant
09. FOOTER       Minimal: brand, legal, contact
```

## Directory / Marketplace

```
01. NAV          Logo + search bar + category nav + post listing CTA
02. HERO         Search-first. Large search input centered. Category pills below.
03. FEATURED     3-4 featured listings. Highlighted visually.
04. CATEGORIES   Icon grid. 6-12 categories. Tap → filtered results.
05. LISTINGS     Card grid (3 cols desktop, 1 col mobile). Filter sidebar left.
        Card: image/icon + name + category tag + rating + CTA
06. HOW IT WORKS 3 steps for both sides (list + discover)
07. NEWSLETTER   "Get new listings weekly" signup
08. FOOTER       Full links + categories repeat + legal
```

## Portfolio / Agency

```
01. NAV          Minimal. Logo + 4 links max + contact CTA
02. HERO         Name / Studio name large. Tagline. "Available for work" badge.
        Selected client logos or work preview background.
03. WORK         Full-bleed case study grid. 4-6 projects.
        Card: image + client + project type. Hover: title reveal.
04. SERVICES     2-3 core offerings. Icon + title + 1-line description.
05. ABOUT        Split: photo left, bio + values right. Human, not corporate.
06. TESTIMONIALS 2-3 client quotes. Name + company + result.
07. CONTACT      Single CTA. Email or form. No phone unless needed.
08. FOOTER       Minimal. Copyright + social icons.
```

## E-Commerce Homepage

```
01. NAV          Logo + category nav + search icon + cart icon + account
02. HERO         Hero product or campaign. Full-bleed. Headline + shop CTA.
03. CATEGORIES   Icon/image grid. 4-6 top categories.
04. FEATURED     "Best sellers" or "New arrivals" product grid (4-8 items).
        Card: image + name + price + quick add
05. BRAND STORY  2-col: lifestyle image + brand values copy. 1 CTA.
06. SOCIAL PROOF Reviews count + star rating + 3 customer photos/quotes
07. NEWSLETTER   Offer-gated: "10% off your first order"
08. FOOTER       4-col: shop, info, social, legal
```

## Course / LMS Landing Page

```
01. NAV          Logo + course links + enroll CTA
02. HERO         Transformation promise (before → after). Instructor photo.
        Enrollment CTA + "X students enrolled" social proof.
03. OUTCOMES     "After this course, you will..." — 4-6 bullet outcomes
04. CURRICULUM   Accordion. Module-by-module. Lesson count per module.
05. INSTRUCTOR   Bio + photo + credentials + 1 achievement
06. TESTIMONIALS 3 student results (specific: "I got hired in 30 days")
07. PRICING      1-2 tiers. What's included. Payment plan option.
08. FAQ          Course-specific: length, prerequisites, certificate, refund
09. FINAL CTA    Urgency: cohort dates or enrollment close
10. FOOTER       Course links + instructor social + legal
```

## Artist EPK / Music Portfolio

```
01. NAV          Artist name/logo + links: Music, Shows, About, Press, Contact
02. HERO         Full-bleed press photo. Artist name large. Genre + location.
        Stream + Book CTA
03. LATEST       Featured release. Album art + streaming links.
04. BIO          Short (150 words max). 3rd person. Achievements + sound.
05. MUSIC        Embedded Spotify/SoundCloud player OR track grid
06. SHOWS        Upcoming dates table. Date + venue + city + ticket link
07. PRESS        Logo strip (media covered) + pullout quote
08. PHOTOS       3-col grid. Press-ready. Download link.
09. CONTACT      Booking email + management + label (if applicable)
10. FOOTER       Social icons + streaming platform icons
```

## CRM / App Dashboard Shell

```
LAYOUT:
  Sidebar (220px fixed left): Logo + nav items + user avatar + signout
  Main content area: Header + page content
  Header: Page title + breadcrumb + action button (right)

DASHBOARD HOME:
  Stats row: 4 KPI cards
  Chart area: Primary chart (full width or 2/3)
  Secondary: Activity feed or table (1/3)
  Quick actions: Floating or pinned button group

STANDARD PAGES:
  List page: Search + filters + data table + pagination
  Detail page: Header + 2-col (main content left, sidebar right)
  Form page: Centered, max-width card, field groups, save/cancel
  Empty state: Centered icon + message + CTA
```
