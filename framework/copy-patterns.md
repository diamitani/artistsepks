# Copy Patterns
## Headline formulas, CTA copy, and section text per industry

---

## Headline Formulas

### SaaS
```
[Outcome] without [pain]
"Run your music career without hiring a team"

[What it is] for [who it's for]
"The business OS for independent artists"

[Number] [thing] in [timeframe]
"Your professional EPK in 10 minutes"

Stop [bad thing]. Start [good thing].
"Stop doing everything alone. Start running your career like a business."

The [thing] [your ICP] actually deserves
"The music business platform independent artists actually deserve"
```

### E-Commerce
```
[Quality claim] + [category]
"Premium sound. Uncompromising quality."

[Feeling]-first copy
"Wear what you mean."

Short. Powerful. Product.
"Sound different."
```

### Portfolio / Agency
```
[What you do] + [for whom]
"We build GTM systems for growth-stage companies."

[Adjective] [work type]
"Thoughtful digital work."

[Proof] + [offer]
"12 years. 200+ projects. Still curious."
```

### Course
```
[Transformation promise]
"Go from broke artist to self-managed professional in 30 days."

[Outcome] without [prerequisite they think they need]
"Master your music business without a manager or lawyer."

[Before state] → [After state]
"From confused to confident: the music business course artists need."
```

## CTA Copy (by goal)

| Goal | CTA Options |
|------|------------|
| Free signup | "Start free" / "Try it free" / "Get started" / "Create your account" |
| Paid purchase | "Get access" / "Join now" / "Claim your spot" / "Start today" |
| Waitlist | "Join the waitlist" / "Get early access" / "Reserve your spot" |
| Download/asset | "Download free" / "Get the template" / "Grab it free" |
| Book/schedule | "Book a call" / "Schedule a demo" / "Talk to us" |
| Learn more | "See how it works" / "Watch the demo" / "Explore features" |

**Never use:** "Click here" / "Submit" / "Learn more" alone / "Get in touch"
**Always be specific:** "Start your free EPK" > "Get started"

## Feature Descriptions (benefit-first formula)

```
WRONG: "AI-powered system prompt builder"
RIGHT: "Build agents that actually know your business"

WRONG: "Knowledge base document storage"
RIGHT: "Your agents remember everything — ICP research, brand voice, past decisions"

WRONG: "Multi-platform DM templates"
RIGHT: "Outreach that sounds human, gets replies, and never gets you banned"

Formula: [What the user CAN DO] not [what the feature IS]
```

## Problem Section Copy

```
Formula: Name the pain in their words, not yours.

Pattern:
"You're doing [specific manual task] when you should be doing [the real work].
[Pain consequence — what it costs them]
[Agitation — why it keeps happening]
[Bridge — "There's a better way"]"

Example (Artispreneur):
"You're your own manager, accountant, lawyer, and publicist.
Every hour spent on paperwork is an hour you didn't spend on the music.
And it's not getting easier — the more your career grows, the more admin follows.
There's a better way."
```

## FAQ Formulas

Always address these objections in order:
1. "Is this worth paying for?" (value)
2. "Will this actually work for me?" (fit)
3. "How long does it take to set up?" (effort)
4. "What if I want to cancel?" (risk)
5. "Is my data safe?" (trust)
6. Specific product objection (unique to product)
7. "Who built this / is this legit?" (credibility)
