# ArtistEPKs.com Site Map

## Pages
| Route | Description |
|---|---|
| `/` | Marketing landing page |
| `/epk/[slug]` | Public EPK page |
| `/[username]` | Public artist profile |
| `/builder` | AI EPK builder |
| `/builder/intake` | Intake wizard |
| `/dashboard` | User dashboard |
| `/dashboard/domains` | Custom domain management |
| `/profile-wizard` | Quick profile creation |
| `/auth/login` | Login |
| `/auth/signup` | Signup |

| Method | Route | Description |
|---|---|---|
| POST | `/api/agent` | AI chat (SSE) |
| GET | `/api/epk` | List EPKs |
| POST | `/api/epk` | Create EPK |
| PATCH | `/api/epk/[id]` | Update EPK |
| DELETE | `/api/epk/[id]` | Delete EPK |
| GET | `/api/profile` | Get profile |
| POST | `/api/profile` | Create/update profile |
| DELETE | `/api/profile` | Delete profile |
| POST | `/api/upload` | Upload image |
| POST | `/api/generate` | AI bio generation |
| GET | `/api/spotify` | Spotify data |
| GET | `/api/social` | Social scraper |
| POST | `/api/social/dashboard` | Social dashboard |
| GET | `/api/pexels` | Photo search |
| GET | `/api/venues` | Venue search |
| GET/POST/DELETE | `/api/domains` | Custom domains |

| POST | `/api/pdf/render` | PDF generation |
